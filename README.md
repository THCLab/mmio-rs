# MMIO – MultiModal Integration Object

## CLI Installation & Usage

### Build

```bash
cargo build --release
```

The binary will be at `target/release/m2io`.

### Install

```bash
cargo install --path .
```

---

### Run

```bash
target/release/m2io --help
```
